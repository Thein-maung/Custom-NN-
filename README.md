# Entangle Chat-2

Quantum-inspired secure PWA for private text & voice chat. Twin AIs simulate entanglement to generate shared OTP pads via QR pairing. No phone number, no server logs, no metadata.

## Features
- Offline QR pairing (entanglement)
- AI-generated OTP pads (neural network)
- E2EE text (quantum OTP XOR)
- P2P voice (WebRTC + OTP chunks)
- Offline-first (SW cache)
- Mobile debug console
- No external dependencies
